1) Установить node.js: https://nodejs.org/en/download/
2) Скачать проект: https://github.com/velskboy/Cypress_autotest
3) Открыть проект в IDE (vs code и т.п.)
4) В терминале зайти в папку со скаченным с GitHub репозиторием
5) Перейти в терминале в папку репозитория new_cypress-main
6) В терминале в папке с проектом запустить: npm install --save-dev cypress@12.7.0
7) В папке с проектом в терминале запустить: npm i
8) Запустить автотесты в терминале командой: npx cypress run
